## 软件设置
- 只接收【验证码类短信】
- 权限设置
    - 发送短信
    - 读取短信
        - 通知类短信
    - 获取手机信息
        - 始终允许
        - 不能设置【空白通行证】，会获取不到卡槽编号，导致不能关联对应的手机号
    - 常驻通知
    - 省电策略为无限制
