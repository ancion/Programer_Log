# 怎样使用搜索引擎

- 视频

## 国内
- 不要使用百度
- 使用微软bing
    - https://cn.bing.com/
    - 能搜到大部分信息

- 其他搜索引擎
    - 平时我都不用
    - 只能搜索到部分信息
    - 搜狗
        - 用来搜微信公众号
    - 360
        - 绑定在360浏览器，设置其他搜索引擎非常麻烦，让你放弃，老老实实使用360搜索

## 国外
